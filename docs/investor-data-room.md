# Investor Data Room Checklist

## Overview

A data room is a secure repository of documents that investors review during due diligence. Having these materials prepared signals professionalism and speeds up fundraising.

---

## Data Room Platforms

### Free Options
- Google Drive (share folder with permissions)
- Notion (investor portal template)
- Dropbox

### Professional Options ($50-$500/month)
- DocSend (tracking, watermarking)
- Carta (cap table + data room)
- Visible (investor updates + data room)
- Dealroom

**Recommendation:** Start with Google Drive or Notion. Move to DocSend when actively raising.
